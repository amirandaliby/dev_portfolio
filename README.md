# Technical Portfolio

Personal portfolio site for Amir Andaliby, built with Next.js and Tailwind CSS.

This version has been customized around software architecture, DevOps, education, IEEE leadership, and technical project work. It includes redesigned sections for the hero, about, experience, skills, projects, education, contact, and footer.

## Highlights

- Modern Next.js portfolio built with the App Router
- Custom experience timeline with alternating cards
- Editorial-style About, Skills, Projects, and Education sections
- Updated branding, favicon, header, footer, and contact experience
- Custom skills icon system with additional SVG assets
- Contact form backed by the existing API route

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Sass
- React Icons
- React Fast Marquee
- Axios
- Nodemailer

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
